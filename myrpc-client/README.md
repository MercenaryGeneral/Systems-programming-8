# myrpc-client 
В этой папке содержится код программы клиента для взаимодействия с myrpc-server. </br>
Данный клиент может отправлять запросы как в TCP, так и в UDP.</br>
## Сборка клиента
```bash
cd myrpc-client
make          # Компиляция
make deb      # Создание deb-пакета
make clean    # Очистка
```
